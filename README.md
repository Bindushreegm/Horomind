# Harmo Mind 🌸

**AI Menstrual Health & Wellness Platform**

A privacy-first web application for tracking cycle health, wellness tips, games, and personalized insights.

## Features
- 📊 Dashboard with cycle tracking
- 📅 Calendar view
- 💬 AI Chatbot support
- 🎮 Mini games with rewards
- 🧘 Yoga & wellness
- 🏪 Store with achievements
- ⚙️ Customizable settings

## Privacy
🔒 All data is stored locally in your browser only. Nothing is sent to external servers.

## 🚀 How to Get it LIVE (Full-Stack)

Since this project now has a backend, you cannot use simple static hosting. Follow these 2 easy steps:

### Step 1: Push to GitHub
Open your terminal in this folder and run:
```bash
git add .
git commit -m "Complete Full-Stack Harmo Mind"
git push origin main
```

### Step 2: One-Click Deploy
Go to [Vercel.com](https://vercel.com) or [Render.com](https://render.com), sign in with GitHub, and select this repository. 
- **Vercel**: It will automatically detect the `vercel.json` and deploy.
- **Render**: Select **Web Service**, it will detect `render.yaml` and deploy.

---

## 🛠️ Local Development
To run the server and frontend together locally:
1. Run `npm install`
2. Run `npm start`
3. Visit `http://localhost:3000`

---

## 🔒 Privacy & Persistence
Data is now stored in a server-side `db.json` database, ensuring your logs are safe even if you clear your browser cache!
